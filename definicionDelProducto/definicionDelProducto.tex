Una plataforma en la Nube para la administración y localización de estacionamiento (SAE) capaz de ajustarse a las necesidades de las diferentes modalidades de estacionamientos  públicos y privados del Distrito  Federal como son estacionamiento, parquímetros, valet-parking y  pensiones.  Permitirá  gestionar  y  realizar  las  tareas operativas y administrativas que se realizan de manera general en un estacionamiento. Figura \ref{fig:ArquitecturaGeneralSAE}

\begin{figure}[h]
	\centering
	\includegraphics[scale=.4]{./definicionDelProducto/source/arquitecturaSAE.pdf}
	\caption{Arquitectura general SAE}
	\label{fig:ArquitecturaGeneralSAE}
\end{figure}		 

La información de los estacionamientos en operación podrá ser visualizada por los usuarios finales a través de una app móvil de manera fácil. Esta mostrará en un mapa los estacionamientos cercamos, número de cajones disponibles, horarios, modos de cobro, tarifas, y una calificación.Figura \ref{fig:usuarioAppLocalizacion}

 Las principales caracteristicas del la plataforma son:
\begin{itemize}
	\item Configuración física
		\begin{itemize}
			\item Número de niveles
			\item Número de cajones
			\item Cajones para personas con capacidades diferectes.
			\item Ubicacón del estacionamiento 
			\item Horario de operación
			\item Modalidad(Pensión, valet, normal)
			\item Publico o Privado
		\end{itemize}
	\item Control de entradas y salidas
	\item Control de ususarios
	\item Generación de reportes
	\item Consultas
	\item Configuración de tarifas
	\item Agregar servicios adicionales (lavado, aspirado, pulido, Etc.)
	\item API de integración para otros sistemas
	\item App de localizaión de estacionamientos
\end{itemize}
