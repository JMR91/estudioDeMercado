\subsection{RF1}
\paragraph{Capacidad}
El sistema permitirá configurar el número de cajones que estarán en operación, así como los cajones que estarán reservados para personas con capacidades diferentes
\subsection{RF2}
\paragraph{Consultas}
Realizará consultas de entradas-salidas e ingresos por:
\begin{itemize}
	\item Día
	\item Usuario
	\item Fecha
\end{itemize}
\subsection{RF3}
\paragraph{Entradas-Salidas}
Llevará el control óptimo de entradas y salidas de vehículos.
\subsection{RF4}
\paragraph{Modos de cobro}
Manejara los siguientes modos de cobros:
\begin{itemize}
	\item Por hora
	\item Por minuto
	\item Por Estancia
	\item Primeros n minutos, con incremento cada n minutos
	\item Boleto perdido
	\item Boleto no sellado
	\item Sin cobro
\end{itemize}
\subsection{RF5}
\paragraph{Historial}
Llevará un historial donde se registrarán los movimientos realizados en el estacionamiento, estos detallaran los cajones, hora de entrada, hora de salida y la cantidad cobrada.
\subsection{RF6}
\paragraph{Reservaciones}
El usuario final podrá hacer una reservación de un cajón de estacionamiento previamente seleccionado desde la aplicación móvil.
\subsection{RF7}
\paragraph{Historial de usuario}
El usuario final podrá llevar un control sobre los estacionamientos visitados, así como aquellos seleccionados por este como favoritos.
\subsection{RF8}
\paragraph{Busqueda de estacionamientos}
El usuario final podrá seleccionar un radio máximo de metros a la redonda en donde serán visualizados los estacionamientos cercanos.
\subsection{RFU}
	\paragraph{Visualizacion de smartphone}
		El ususario final podrá visualizar en un dispositivo movil(smarphone), los estacionamientos cercanos a su ubicacion, mostrando información del estacionamiento al seleccionarlo. 
		\begin{itemize}
			\item Horario de operacón
			\item Capacidad y cajones disponibles.
			\item Modos de cobro
			\item Servicios adicionales
			\item Tarifas especiales
		\end{itemize}
	
\subsection{RFU}
	\paragraph{Habilitar reservación}
		El administrador podrá habilitar o desbilitar la opción de \textit{Reservacion de Cajón} para su estacionameinto. Este deberá indicar la tolerancia para que el usuario acceda al estacionamiento, en caso contrario perdera su reservación.
