\subsection{Administrador}

	\hrulefill
	\subsubsection{Visivilidad}
		Los administradores podrán elegir si el estacionamiento es visible para todos los usuarios o solo para usuarios finales que el autorice.
		
	\hrulefill	
	\subsubsection{Tipos de estacionamiento}
			\begin{itemize}
				\item \textbf{\textit{Estacionamiento c/s Pensión}} Serán estacionamientos donde los usuarios finales busquen un cajón disponible donde aparcar su vehículo. Al registra un nuevo estacionamiento de este tipo se podrá agrupar los cajones por \textbf{Nivel}(representan los pisos de los estacionamientos que cuenten con más de un nivel) y \textbf{Grupo} (agrupación adicional que sirve para ubicar en que bloque del estancamiento esta el cajón). A cada cajón se le podrá asignar un o varios pensionados, e indicar si estará reservado para discapacitados.
				
				\item \textit{Valet Parking} Es un servicio de aparcamiento que se le da a los usuarios. En este modelo los usuarios dejan su vehículo en recepción y los encargados se ocupan de llevar el vehículo hasta el estacionamiento. Para este tipo de estacionamientos el administrador podrá dar de alta uno o varios estacionamientos, donde, podrán estacionar los vehículos, cada estacionamiento del valet podrá contara con Niveles, grupos y cajones.
				
				\item \textit{Parquímetros} Son los estacionamientos que están en calles públicas y pueden ser utilizados por cualquiera. Para este tipo de estacionamientos el administrador pueda dar de alta zonas. Una zona representa una calle o calles donde se pueden registra el un número cajones para aparcar vehículos.
			\end{itemize}
			
	\hrulefill	
	\subsubsection{Pensionados con múltiples vehículos}
		El administrador podrá indicar si al reservar un cajón a para un pensionado este puede hacer uso de el solo con un vehículo o con varios.
		
	\hrulefill	
	\subsubsection{Forma de estacionamiento}
		El administrador tendrá la opción de cargar una plantilla(croquis) del estacionamiento para representar la distribución del estacionamiento.

	\hrulefill	
	\subsubsection{Consultas}
		Realizará consultas de entradas-salidas e ingresos por:
		\begin{itemize}
			\item Día
			\item Usuario
			\item Fecha
		\end{itemize}

	\hrulefill	
	\subsubsection{Entradas\/Salidas}
		Llevará el control óptimo de entradas y salidas de vehículos.

	\hrulefill	
	\subsubsection{Modos de cobro}
		Manejará los siguientes modos de cobros:
		\begin{itemize}
			\item Por hora
			\item Por minuto
			\item Por Estancia
			\item Primeros n minutos, con incremento cada n minutos
			\item Boleto perdido
			\item Boleto no sellado
			\item Sin cobro
		\end{itemize}

	\hrulefill	
	\subsubsection{Tarifas especiales}
		El sistema permitirá agregar precios especiales como promociones para clientes.

	\hrulefill	
	\subsubsection{Historial}
		Llevará un historial donde se registrarán los movimientos realizados en el estacionamiento, estos detallaran los cajones, hora de entrada, hora de salida y la cantidad cobrada.
	
	\hrulefill	
	\subsubsection{Habilitar reservación}
		Podrá habilitar o deshabilitar la opción de \textit{Reservacion de Cajón} para sus estacionamientos. En caso de habilitarlo deberá indicar la tolerancia para que el usuario acceda al estacionamiento.

	\hrulefill	
	\subsubsection{Control de usuarios}
		Podrá gestionar las cuenta de los operadores.
	
\subsection{Usuario Final}

	\hrulefill	
	\subsubsection{Reservaciones}
		Podrá hacer una reservación de un cajón de estacionamiento previamente seleccionado desde la aplicación móvil.
	
	\hrulefill	
	\subsubsection{Historial de usuario}
		Podrá llevar un control sobre los estacionamientos visitados, así como aquellos seleccionados por este como favoritos.
	
	\hrulefill	
	\subsubsection{Búsqueda de estacionamientos}
		Podrá seleccionar un radio máximo de metros a la redonda en donde serán visualizados los estacionamientos.
	
	\hrulefill	
	\subsubsection{Visualización de smartphone}
		Podará visualizar en un dispositivo móvil(smartphone) los estacionamientos registrados cercanos a su ubicación , mostrando información del estacionamiento al seleccionarlo. 
		\begin{itemize}
			\item Horario de operación
			\item Capacidad y cajones disponibles.
			\item Modos de cobro
			\item Servicios adicionales
			\item Tarifas especiales
		\end{itemize}
