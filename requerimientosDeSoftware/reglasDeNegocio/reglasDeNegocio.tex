\hrulefill
\subsubsection{Reservación}
	Para que el usuario final pueda reservar un cajón es necesario que este dado de alta.   

\hrulefill
\subsubsection{Servicios adicionales}
	Se podrán gestiosnar servicios adicionales(lavado, aspirado, pulido, Etc.). Estos podrán tener un cargo extra(si así lo desea el administrador).

\hrulefill
\subsubsection{Número de accesos}
	Los estacionamientos podrán tener de uno a muchos accesos. Estos accesos es donde se registran entradas y salidas.

\hrulefill
\subsubsection{Alta vehículo}
	Para registrar un vehículo se solocitaran los siguientes datos:
	\begin{itemize}
			\item Placa
			\item Marca
			\item Modelo				
			\item Color
			\item Detalles(opcional)				
			\item Fotos(opcional)
		\end{itemize}

\hrulefill
\subsubsection{Alta usuario App}
	Los usuarios se podrán registrar desde la app móvil. Los datos solicitados son:
	\begin{itemize}
		\item Nombre completo
		\item Teléfono
		\item Móvil
		\item e-mail
		\item Dirección 
	\end{itemize}	

\hrulefill
\subsubsection{Control de pensionados}
	Se podrá agregar multiples pensiones a un cajon siempre y cuando no se traslape los horarios de uso. En horarios donde no este asignado un pensionado ese cajon estara libre para su uso.

\hrulefill
\subsubsection{Control de pensionados}
	Se podrá agregar multiples pensiones a un cajon siempre y cuando no se traslape los horarios de uso. \\Se podrá usar un cajón siempre y cuando no este en horario de pensión. En caso de que entre en uso el horario de pensión y no el cajon este ocupado por otro vehículo se tendrá que reservar un cajon povisional. 
	
\hrulefill
\subsubsection{Alta Pensionado}
	Para dar de alta un pensionada se deberá dar los siguentes datos.
	\begin{itemize}
		\item Nombre completo
		\item Teléfono
		\item Móvil
		\item e-mail
		\item Dirección
		\item Placa del vehiculo registrado
	\end{itemize}
	Otra opcion será vincular con un usuario registrado.

\hrulefill	
\subsubsection{Registro valet}
	Al recibir un vehículo el operador podrá indicar el estado como se resive el vehículo(con rayaduras, golpes, fata de alguna pieza, Etc.).
	
\hrulefill
\subsubsection{Horario de pensión}
	Para dar de alta un pensionado, el usuario final debera estar dado de alta en el sistema como pensionado.
	\begin{itemize}
		\item Placas del vehículo 
		\item Horario de uso
		\item Vegencia de uso			
	\end{itemize}

\hrulefill
\subsubsection{Tiempo de vida de la información}
	La información será alamcenada en el sistema, pero no seá eliminada.

\hrulefill

\hrulefill
\subsubsection{Alta  estacionamiento} \label{rn:altaEstacionamiento}
	Los datos que sin importar el tipo de estacionamiento se solicitaran son:
	\begin{itemize}
		\item Nombre 
		\item Logo de la empresa o del estacionamiento(opcional)
		\item Horario de operación
		\item Servicios adicionales
		\item Tarifas 
		\item Visivilidad (público o privado)
		\item Estabecimientos
	\end{itemize}
	Es posible configurar 3 tipos de estacionamientos. 
	\begin{enumerate}
		\item \textit{Valet} 
			\begin{itemize}
				\item Número de lugares(estacionamientos) donde se podrán aparcar vehículos,así como, su ubicación. 
				\item Número de niveles cajones
				\item Número de cajones por nivel
				\item Indicar si se los usuarios finales podrán solicitar con anticipación su vehículo desde su dispositivo móvil.   
			\end{itemize}
		\item \textit{Estacionamiento / Pensión} 
			\begin{itemize}
				\item Ubicaión
				\item Número de niveles cajones
				\item Número de cajones por nivel
				\item Cajones reservados para dicapaciatos
				\item Indicar si algún cajón tendrá pensión
			\end{itemize}
		\item \textit{Parquímetro}
			\begin{itemize}
				\item Número de zonas con su respectiva área
			\end{itemize}
	\end{enumerate}
